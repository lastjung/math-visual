# Polygon Sound 추가 기능 아이디어

다음은 Polygon Sound 프로젝트에 추가 가능한 기능 목록입니다.

1. **스케일 프리셋 (Scale Presets)**
   - Pentatonic 외에 Dorian, Phrygian, Blues, Whole-tone, Chromatic 등 다양한 스케일 선택 기능 추가.
   - **(구현 완료)** 현재 Pentatonic, Dorian, Phrygian, Blues, Whole-tone 5가지 스케일이 적용되었습니다. 상세 설명은 하단을 참조하세요.

2. **루트 키 변경 (Root Key Change)**
   - 현재 고정된 C 키(**C Major**) 대신 루트 노트를 변경하여 조옮김(Transposition) 기능 제공.
   - 예: D Minor Pentatonic, F Lyxian 등.

3. **템포 스냅 (Tempo Snap)**
   - 부드러운 속도 조절 외에 음악적인 BPM 단위(60, 90, 120, 150 BPM)에 딱 맞춰지는 'BPM Snap' 모드 지원.
   - 리듬감 있는 폴리리듬 생성에 도움.

4. **잔향 스타일 (Ambience / Reverb Styles)**
   - 딜레이/리버브 설정을 프리셋으로 제공.
   - **Soft**: 짧고 따뜻한 잔향
   - **Space**: 길고 몽환적인 우주 느낌 (현재 딜레이 강화)
   - **Glass**: 맑고 깨끗한, 짧은 딜레이

5. **레이어 오케스트레이션 (Layer Orchestration)**
   - 레이어(L-1, L-2...) 별로 다른 음색이나 옥타브 할당.
   - 예: L-1은 베이스(낮음), L-2는 멜로디(중간), L-3는 차임(높음/반짝임).

6. **시각-사운드 동기화 (Visual-Audio Sync)**
   - 활성 노트가 연주될 때, 단순히 점이 커지는 것을 넘어 짧은 링(파동)이 퍼져나가는 애니메이션 추가.
   - **(구현 완료)** 선택된 스케일의 노트가 연주될 때 링 애니메이션이 적용됩니다.

7. **무드 모드 (Mood Mode)**
   - 색상 테마 + 스케일 + 사운드 설정을 한 번에 변경하는 '원터치 무드' 버튼.
   - **Dream**: 파스텔 톤 + 메이저 펜타토닉 + 긴 리버브
   - **Ritual**: 붉은/금색 톤 + 프리지안 스케일 + 드럼/타악기 느낌
   - **Lullaby**: 밤하늘 톤 + 멜로딕 마이너 + 오르골 사운드

8. **녹음/리플레이 (Loop Recorder)**
   - 사용자가 만든 패턴을 10~20초 정도 녹음하여 루프(Loop)로 지정.
   - 배경에 깔아두고 그 위에 실시간으로 연주하는 기능.

# 스케일 상세 설명 (Scale Details)

다음은 구현된 5가지 스케일 프리셋의 특징과 음악적 분위기에 대한 상세 설명입니다.

## 1. Pentatonic Scale (펜타토닉 스케일)

**구성음 (C Major):** C, D, E, G, A

- **특징:** 반음이 포함되지 않은 5음계로, 가장 보편적이고 안정적인 소리를 냅니다. 불협화음이 거의 없어 어떤 순서로 연주해도 자연스럽게 어울립니다.
- **분위기:** 순수함, 평화로움, 동심, 전원적인 느낌. 민요나 자장가에서 자주 쓰입니다.

## 2. Dorian Mode (도리안 모드)

**구성음 (C Root):** C, D, Eb, F, G, A, Bb

- **특징:** 단조(Minor) 기반이지만 6번째 음이 반음 올라가 있어(Natural 6th), 일반적인 단조보다 밝고 세련된 느낌을 줍니다.
- **분위기:** 신비로움, 희망찬 슬픔, 몽환적, 도시적인 밤. 재즈나 켈트 음악, 판타지 영화 OST에 자주 등장합니다.

## 3. Phrygian Mode (프리지안 모드)

**구성음 (C Root):** C, Db, Eb, F, G, Ab, Bb

- **특징:** 시작하자마자 반음(Minor 2nd)이 등장하여 강한 긴장감과 이국적인 색채를 만듭니다. 스페인 플라멩코 음악의 핵심 스케일입니다.
- **분위기:** 정열적, 긴장감, 어두움, 이국적, 고대 의식. 강렬하고 드라마틱한 분위기를 연출할 때 좋습니다.

## 4. Blues Scale (블루스 스케일)

**구성음 (C Root):** C, Eb, F, Gb, G, Bb

- **특징:** 마이너 펜타토닉에 '블루 노트'(불협화음인 감5도, Gb)를 추가한 6음계입니다. 이 블루 노트가 묘한 슬픔과 끈적한 느낌을 만듭니다.
- **분위기:** 우울함, 소울풀(Soulful), 끈적함, 감성적. 재즈와 블루스 음악의 기본이 되는 스케일입니다.

## 5. Whole-tone Scale (온음 스케일)

**구성음 (C Root):** C, D, E, Gb, Ab, Bb

- **특징:** 모든 음의 간격이 온음(Whole step)으로만 이루어진 6음계입니다. 중심음(Tonic)이 뚜렷하지 않아 시작과 끝이 없는 듯한 모호한 느낌을 줍니다.
- **분위기:** 꿈속, 무중력 상태, 회상, 신비로움, 흐릿함. 드뷔시(Debussy) 같은 인상주의 음악이나 영화의 회상 씬에서 자주 들을 수 있습니다.
