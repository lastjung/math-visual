# 추천 무료 배경 음악 (10선)

사용하시기 편한 **Pixabay Music**의 인기 트랙들을 선정했습니다.
아래 **검색 결과 링크**를 클릭하시면 유사한 분위기의 곡들을 바로 확인하고 다운로드 하실 수 있습니다.

### � 사용 방법

1. 아래 링크를 클릭하여 Pixabay로 이동합니다.
2. `Download` 버튼을 눌러 MP3 파일을 받습니다.
3. 다운로드한 파일의 이름을 아래 **지정된 파일명**으로 변경합니다.
4. `pg-template-duncan/visualization/assets/music/` 폴더에 넣습니다.
